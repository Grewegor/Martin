# 🎵 Сайт "МАРТИН" - Краткое руководство

## ✅ Все требования выполнены

### 1. **Авторизация пользователей**
- ✓ Страница входа: `login.html`
- ✓ Демо учетные данные: `demo / demo123`
- ✓ Защита всех страниц от неавторизованного доступа
- ✓ Токены сохраняются в localStorage

### 2. **Регистрация новых пользователей**
- ✓ Форма регистрации на `login.html`
- ✓ Валидация на клиенте:
  - Имя пользователя: 3-20 символов
  - Email: проверка формата
  - Пароль: минимум 6 символов
  - Подтверждение пароля
- ✓ Сохранение в `data/users.json`
- ✓ Проверка уникальности имени и email

### 3. **Защищенный контент**
- ✓ Главная страница (`index.html`) - только для авторизованных
- ✓ Страница тура (`tour.html`) - только для авторизованных
- ✓ Автоматический редирект на login.html для неавторизованных

### 4. **Таблица с данными (Bootstrap-table стиль)**
- ✓ Таблица альбомов на главной странице
- ✓ Данные из `data/albums.json`
- ✓ Колонки: Год, Название альбома, Действия
- ✓ Кнопка удаления для каждой записи

### 5. **AJAX для загрузки/сохранения**
- ✓ Загрузка альбомов: `fetch()` → `data/albums.json`
- ✓ Добавление альбома: `fetch()` POST → `api/save_albums.php`
- ✓ Удаление альбома: `fetch()` → обновление JSON
- ✓ Все операции без перезагрузки страницы

### 6. **Валидация на клиенте (без библиотек)**
- ✓ Имя пользователя (regex)
- ✓ Email (regex)
- ✓ Пароль (длина)
- ✓ Год альбома (4 цифры, 1900-2100)
- ✓ Название альбома (2-100 символов)

## 🔐 Система авторизации

### API endpoints

| Метод | Endpoint | Описание |
|-------|----------|---------|
| POST | `/api/login.php` | Вход в систему |
| POST | `/api/register.php` | Регистрация нового пользователя |
| GET | `/api/check_auth.php` | Проверка валидности токена |
| POST | `/api/logout.php` | Выход из системы |

### Формат токена
- Генерируется как random_bytes(32)
- Сохраняется в `localStorage`
- Срок действия: 7 дней
- На сервере хранится в `data/tokens/`

## 📊 Управление альбомами

### Просмотр
- Таблица загружается при загрузке страницы
- AJAX запрос: `fetch('data/albums.json')`
- Автоматическое обновление после добавления/удаления

### Добавление
```javascript
// Запрос
GET /api/save_albums.php?year=2026&title=New Album

// Ответ
{ "success": true }
```

### Удаление
```javascript
// Запрос
POST /api/save_albums.php
Body: { "albums": [...] }

// Ответ
{ "success": true }
```

## 🧪 Тестирование

### Тест авторизации
1. Откройте `http://localhost/lab4`
2. Должен быть редирект на `login.html`
3. Введите: `demo / demo123`
4. Должна открыться главная страница

### Тест регистрации
1. На странице входа нажмите "Регистрация"
2. Попробуйте заполнить неправильные данные:
   - Имя < 3 символов → ошибка
   - Некорректный email → ошибка
   - Пароль < 6 символов → ошибка
   - Пароли не совпадают → ошибка

### Тест альбомов
1. После входа отобразится таблица альбомов
2. Добавьте альбом:
   - Год: `2026`
   - Название: `Тестовый альбом`
   - Нажмите "Добавить"
3. Альбом должен появиться в таблице
4. Нажмите "Удалить" рядом с альбомом
5. Подтвердите удаление
6. Альбом должен исчезнуть
